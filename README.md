# PTTscrawler
Using python to scrawl ptt


# Here is the way to operate this script
> python pttscrawler 板名 搜尋頁數

# 內容格式
> a_num: 文章編號(第幾篇文章) <br>
> b_author: 文章作者 <br>
> c_title: 文章標題 <br>
> d_date: 發文日期 <br>
> e_like: 文章推文個數 <br>
> f_hate: 文章噓文個數 <br>
> g_comment: 文章留言個數(不是推文和噓文) <br>
> h_content: 文章內容 <br>
> i_message: 留言內容 <br>
